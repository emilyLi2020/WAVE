# Wave

Hackathon repo: Cursor skills live under `.agents/skills/`. The runnable app is in **`web/`** (Next.js + TypeScript + Tailwind).

## Run the app

```bash
cd web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Use **Open core feature** or the nav to visit `/feature`.
